# DVAPI-Report

DVAPI is a small intentionally vulnerable API designed for learning and practicing API security testing.  
It contains common flaws like weak authentication, insecure object access, and basic injection issues.

## Features
- Simple login and user endpoints
- Common API vulnerabilities to practice
- Easy to run with a single command
